# adopet

# Sobre esse projeto
O Adopet é uma plataforma fictícia para conectar pessoas que desejam adotar animais de estimação e abrigos. Este projeto representa o back-end da plataforma.

# Tecnologias utilizadas
- Java 17
- Spring Boot (3.1.0)
- Spring Data JPA
- Spring Security
- Maven
- JUnit 5 
- Postgres DB
